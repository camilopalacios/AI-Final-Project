Welcome Page

The purpose of this webpage is to summarize the developement of my AI project on the RTS game Starcraft Broodwar. The project focuses on the combat aspect of the game and aims to improve the default AI in a general combat situation.

Content:

  • Q-Learning in a grid world:

    In this experiment a Dragoon tries to find the goal state at the end of a corridor surrounded by dead pits.

    Description:

    The agent (the Dragoon) in this grid world is a Protoss Dragoon. The agent's state is defined in terms of its possition in the map.
